# Projeto Android Mascote

Página web estática construída em HTML e CSS que conta a história do mascote do Android, o Bugdroid.

## Descrição

O projeto apresenta um artigo informativo sobre a origem do mascote do Android, incluindo:
- a primeira versão criada por Dan Morrill;
- a criação do Bugdroid por Irina Blok;
- curiosidades sobre as versões do Android com nomes de doces;
- imagens e vídeo incorporado.

## Estrutura do projeto

- `index.html` - página principal do site.
- `style.css` - estilos visuais usados pelo site.
- `pacote-projeto-d010/` - recursos do site, como imagens, fontes e arquivo auxiliar.
  - `imagens/` - imagens usadas na página.
  - `fontes/` - fonte personalizada usada no design.
  - `android-site.txt` - arquivo de texto adicional.

## Como usar

1. Abra `index.html` em um navegador.
2. Verifique se a pasta `pacote-projeto-d010` está junto com `index.html` para que imagens e fontes carreguem corretamente.

## Observações

- O site usa um vídeo incorporado do YouTube para complementar o conteúdo.
- O projeto foi criado no contexto de um desafio de HTML e CSS.
